# Capturas de Pantalla

## Instrucciones para agregar capturas

1. Ejecutar el frontend: `npm run dev`
2. Tomar las siguientes capturas:
   - `productos-listado.png` - Vista principal con productos cargados
   - `venta-exitosa.png` - Pantalla de confirmación de venta
   - `error-api-caido.png` - Manejo de error cuando API no responde

3. Guardar las imágenes en esta carpeta

## Capturas Requeridas

### 1. productos-listado.png
- Mostrar barra de búsqueda
- Lista de productos con nombre, precio y stock
- Botón "Agregar" visible

### 2. venta-exitosa.png
- ID de venta generado
- Cálculo de cambio
- Botones de imprimir recibo y nueva venta

### 3. error-api-caido.png
- Banner de error visible
- Mensaje descriptivo
- Botón de reintentar
